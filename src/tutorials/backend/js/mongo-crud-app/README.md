# Backend API with NodeJS, Express and MongoDB

## Installation

### Install Express
Open terminal and write the following command
```
npm install express
```

### Install Nodemon
Open the terminal and write the following command
```
npm i nodemon
```

### Install Mongodb
Open the terminal and write the following command
```
npm i mongodb
```

### Install Mongoose
Open the terminal and write the following command
```
npm install mongoose
```
